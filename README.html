# atividade-github Inicializando um Repositório Local: Antes de começar a usar o Git, você precisa iniciar um repositório local em seu sistema. Para fazer isso, navegue até o diretório do seu projeto usando o terminal ou prompt de comando e execute o comando: git init Isso criará um repositório local em seu computador. Trabalhando com Arquivos: Adicione arquivos ao repositório com o comando: git add nome_do_arquivo ou para adicionar todos os arquivos: git add. Criando um Commit: Após adicionar arquivos, você precisa criar um commit. Um commit é uma operação que agrupa um conjunto de alterações feitas em arquivos e os adiciona ao histórico do repositório.
