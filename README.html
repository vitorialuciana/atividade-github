# atividade-github Inicializando um Repositório Local Antes de começar a usar o Git, você precisa iniciar um repositório local em seu sistema. Para fazer isso, navegue até o diretório do seu projeto usando o terminal ou prompt de comando e execute o comando: git init Isso criará um repositório local em seu computador. Trabalhando com Arquivos: Adicione arquivos ao repositório com o comando: git add nome_do_arquivo ou para adicionar todos os arquivos: git add. Criando um Commit: Após adicionar arquivos, você precisa criar um commit. Um commit é uma operação que agrupa um conjunto de alterações feitas em arquivos e os adiciona ao histórico do repositório. Use o seguinte comando: git commit -m "Descrição do commit Verificando o Status do Repositório: A qualquer momento, você pode verificar o estado atual do seu repositório com o comando: git status Visualizando o Histórico de Commits: Você pode visualizar o histórico de commits com: git log Branches: Branches são ramificações do seu repositório que permitem que você trabalhe em novas funcionalidades ou correções de bugs sem afetar o código principal. Para criar um novo branch: git branch nome_do_branch Para mudar para esse branch: git checkout nome_do_branch Clonando um Repositório Remoto: Se você quiser criar uma cópia local de um repositório remoto, você pode cloná-lo usando: git clone url_do_repositorio Trabalhando com Repositórios Remotos: Para adicionar um repositório remoto: git remote add origin url_do_repositorio Para enviar commits para o repositório remoto: git push origin nome_do_branch Para receber commits do repositório remoto: git pull origin nome_do_branch Fusão de Branches: Quando você terminar de trabalhar em uma funcionalidade ou correção de bug em um branch, você pode mesclar suas mudanças de volta ao branch principal usando: git merge nome_do_branch Resolução de Conflitos: Às vezes, durante a fusão de branches, pode haver conflitos. Você precisará resolver esses conflitos manualmente, editando os arquivos conflitantes e, em seguida, fazendo o commit das mudanças.
